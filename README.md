# TicTacToe

A native Android Tic-Tac-Toe game, built with Kotlin and Jetpack Compose.

## Tech Stack

- **Language:** Kotlin
- **UI:** Jetpack Compose, Material 3
- **Min SDK:** 24 · **Target SDK:** 37

## Status

🚧 Early scaffold — the project currently has the base Compose activity set up, but the
game board, turn logic, and win/draw detection haven't been implemented yet.

## Roadmap

- [ ] 3x3 game board UI
- [ ] Turn-based X/O placement logic
- [ ] Win and draw detection
- [ ] Reset / play-again flow
- [ ] Score tracker across rounds
- [ ] Single-player mode vs. a simple AI opponent

## Getting Started

1. Clone the repo:
   ```
   git clone https://github.com/haidershah/TicTacToe.git
   ```
2. Open the project in Android Studio.
3. Run on an emulator or physical device (API 24+).
